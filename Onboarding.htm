Create Virtual Machine and Install/Configure Your Initial System

·         Reference & watch the “General” section training videos.

·         Create a Virtual Machine in our Data Center (SCVMM Install guide and best practices)

·         Install PI Data Archive & PI AF (scripts!). 

·         Experiment with the system while you watch the training videos to become familiar with the items below.  After working through this section you should be able to give a summary description of the following:

o   PI Data Archive

o   PI AF

o   Connectors

o   Interfaces

o   PI System Management Tools

o   Coresight

o   DataLink 

 

Establish Development Environment

·         Reference our Manufacturing Document from TFS.  It provides the instructions to guide this activity.  This obviously means you’ll first need to get setup with TFS & VSO.

·         Using the version of our system generated on your dev machine, demonstrate that you’re able to visualize PI data on an Esri Map.  There’s a lot of background research & learning you’ll need in order to complete this one:

o   Watch the Esri Integrator videos

o   Read the latest version of the Esri Integrator User’s Guide

o   We hold a lab at our various conferences & the manual for that lab is an excellent resource.  You’ll want to read that as well while you are establishing your environment.

o   Asking questions!
